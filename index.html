Estoque CDL

Para sincronizar o estoque entre todos os funcionários, precisamos conectar ao Supabase — banco de dados gratuito na nuvem.

Configuração inicial — faça isso uma única vez
Encontre em: Project Settings → API → Project URL
Encontre em: Project Settings → API → Project API Keys → anon public
Como criar sua conta gratuita no Supabase:
1. Acesse supabase.com e clique em Start your project
2. Crie uma conta com Google ou GitHub
3. Clique em New Project, dê um nome (ex: estoque-cdl) e crie uma senha
4. Aguarde ~2 minutos enquanto o projeto é criado
5. Vá em SQL Editor e cole e execute o SQL abaixo:

CREATE TABLE estoque ( sku BIGINT PRIMARY KEY, quantidade INTEGER DEFAULT 0 ); CREATE TABLE historico ( id BIGSERIAL PRIMARY KEY, tipo TEXT, sku BIGINT, nome TEXT, qtd INTEGER, data DATE, obs TEXT, colegio TEXT, criado_em TIMESTAMPTZ DEFAULT NOW() );
6. Cole a URL e a chave acima e clique em Conectar
